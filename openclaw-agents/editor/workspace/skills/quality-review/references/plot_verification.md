# 伏笔兑现验证指南

## 核心规则

**伏笔说"A被羞辱/威胁" → 必须是A本人被直接羞辱/威胁**

---

## 三步验证法

### 1. 触发对象验证（最关键！）

**检查方法**：
1. 读取伏笔描述，提取核心条件（谁 + 被怎么样）
2. 在章节内容中找到对应的场景
3. 确认触发对象与伏笔要求一致

**错误示例（S004伏笔）**：

```
伏笔：当赵婉清被羞辱或威胁时，林默会失去理智

❌ 错误兑现：
场景：赵国栋说"你让婉清以后怎么见人"
分析：这是羞辱林默"让赵婉清丢脸"，不是羞辱赵婉清本人
触发对象：林默（错误！）
扣分：-15分（伏笔兑现逻辑不通）

❌ 错误兑现：
场景：赵国栋威胁"赵家的产业，你别想继承一分钱"
分析：这是威胁"继承权"，不是威胁"赵婉清本人"
触发对象：继承权（错误！）
扣分：-15分（伏笔兑现逻辑不通）
```

**正确兑现**：

```
场景：赵国栋直接羞辱赵婉清："婉清，你看看你这个丈夫，
连给你买件像样的衣服都要跟我伸手。你嫁给他三年，
他给你挣过一分钱吗？你是我赵国栋的女儿，嫁给这么一个废物，
你以为你在帮他？你是在毁你自己。"

分析：这是直接羞辱赵婉清本人
触发对象：赵婉清（正确！）
赵婉清的反应：脸色刷地变白，眼眶泛红
林默的反应：眼神瞬间变了——近乎疯狂的、非理性的保护欲
```

---

### 2. 触发条件验证

检查要点：
- 伏笔说"当X时" → 检查章节中X是否明确发生
- ❌ 错误：Y发生，X被暗示
- ✅ 正确：X明确发生

---

### 3. 角色反应验证

检查角色反应是否符合伏笔设定：

```
伏笔：林默会失去理智

❌ 错误：林默只是"握紧拳头，指节发白"
分析：这是愤怒，不是"失去理智"
扣分：-8分（角色反应不符合伏笔）

✅ 正确：林默的眼神变了——近乎疯狂的、非理性的保护欲
分析：这是"失去理智"的体现
```

---

## 验证清单

```
□ 核心条件：伏笔要求的触发条件是否满足？
□ 触发对象：伏笔说的对象是否被正确对待？
□ 角色反应：反应是否合理适度？（不过度、不缺失）
```

## 硬伤处理

**伏笔逻辑硬伤 → 直接退回，总分 ≤ 60分**
