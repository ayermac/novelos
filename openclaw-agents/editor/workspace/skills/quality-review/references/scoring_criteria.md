# 五层审校评分标准

## 评分汇总表

| 检查层 | 权重 | 满分 | 必退红线 | 备注 |
| :--- | :--- | :--- | :--- | :--- |
| **设定一致性** | 25% | 25 | < 18 | 错一项核心数值即退回 |
| **逻辑漏洞** | 25% | 25 | < 18 | 反派智商必须在线 |
| **毒点检测** | 20% | 20 | < 15 | 出现说教、圣母、严重憋屈即退回 |
| **文字/去AI化** | 15% | 15 | < 10 | AI味过浓、排比/复读过多即退回 |
| **爽点与钩子** | 15% | 15 | < 10 | 无末尾钩子必退 |
| **总计** | **100%** | **100** | **总分 < 90** | **任一单项触发必退线也必须退回** |

---

## 一、设定一致性 (0-25分) —— 核心硬指标

**及格线：18分。低于18分直接判定不通过，无论总分。**

| 检查项 | 扣分 | 错误示例/判定标准 |
| :--- | :--- | :--- |
| **字数偏差** | -5~-10 | 与指令要求的字数目标偏差 >30%扣5分，>50%扣10分 |
| **数值/资源漂移** | -5~10 | 上章剩余2点，本章突然变5点且无获得描述；等级乱跳。 |
| **技能/能力突变** | -5 | 未通过系统解锁或学习，突然施展新技能或拥有某种专业背景。 |
| **世界规则违背** | -8 | 职场背景突然出现超自然力；系统规则前后矛盾。 |
| **角色人设崩塌** | -5 | 利益至上的反派突然变温和，或主角性格突变（如从杀伐果断变圣母）。 |
| **时空连续性** | -3 | 瞬移、日期倒流、年龄计算错误、地理位置瞬移。 |
| **状态卡矛盾** | -5 | 本章内容与上一章状态卡描述矛盾 |

---

## 二、逻辑漏洞 (0-25分) —— 智商守护线

**及格线：18分。严禁通过"降智"来强行推动剧情。**

| 检查项 | 扣分 | 说明 |
| :--- | :--- | :--- |
| **伏笔兑现逻辑不通** | -8~-15 | 伏笔要求的核心条件不满足，或触发逻辑不合理。 |
| **反派降智** | -8 | 高管/老狐狸在常识性问题上犯低级错误，强行给主角送人头。 |
| **信息全知视角** | -5 | 主角利用了"设定上他尚未知晓"的信息做决策，缺乏推导过程。 |
| **动机虚假** | -5 | 角色为了剧情需要而行动，缺乏核心利益驱动。 |
| **强行误会** | -4 | 两个角色在该交流时强行"我不听/我不说"制造低级矛盾。 |
| **机械降神** | -6 | 绝境下无铺垫地跳出一个新人物或新金手指救场，毫无逻辑支撑。 |

---

## 三、毒点检测 (0-20分) —— 用户留存线

**及格线：15分。任何严重违规项直接 0 分并退回。**

| 类型 | 扣分 | 示例/判定标准 |
| :--- | :--- | :--- |
| **极致憋屈** | -10 | 主角被反派单方面羞辱超过两章且无任何隐忍或反击预案。 |
| **圣母/双标** | -8 | 无底线原谅仇人；主角可以违规，配角违规就被主角严惩。 |
| **上帝说教** | -5 | AI高发：在段落末尾用上帝视角强行总结人生哲理、职场真谛。 |
| **强行保密** | -4 | 核心秘密该说不说，纯粹为了拖延剧情节奏而刻意隐瞒。 |
| **违规/三观** | -20 | 政治敏感、过度血腥、色情低俗、宣扬扭曲价值观。 |

---

## 四、文字质量与"去AI化" (0-15分)

**及格线：10分。重点打击"公文感"和"机械感"。**

| 类型 | 扣分 | 判定标准 |
| :--- | :--- | :--- |
| **AI式排比/句式** | -3 | 连续3段使用"不仅...而且..."、"既是...又是..."、"仿佛...又像..."。 |
| **动作/神态复读** | -3 | AI高发：一章内出现3次以上的"冷笑"、"嘴角微扬"、"闪过寒芒"。 |
| **空洞万能修辞** | -2 | 大量使用"星辰大海"、"不禁感慨"、"如坠冰窟"等无意义形容词。 |
| **文字灌水** | -2 | 对读者已知的逻辑进行反复解释，或对无关紧要的背景大篇幅灌输。 |
| **基础规范** | -2 | 错别字（如"在/再"不分）、成分残缺、标点符号错误。 |

---

## 五、爽点密度与钩子 (0-15分)

**及格线：10分。结尾必须有钩子，否则判定为断点失败。**

| 检查项 | 权重 | 要求 |
| :--- | :--- | :--- |
| **爽点密度** | 5分 | 每500字必须有：被认可、获资源、小反击、数值涨等正向反馈。 |
| **情绪张力** | 5分 | 必须有"压抑-反弹"的过程，严禁平铺直叙、毫无波澜的胜利。 |
| **末尾钩子** | 5分 | 强制要求：必须留下未解悬念、新危机或重大反转，诱导点开下一章。 |
