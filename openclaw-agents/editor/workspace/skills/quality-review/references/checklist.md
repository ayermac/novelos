# 质检检查清单

## 基础对齐

```
□ 字数统计：实际字数是多少？与目标偏差多少？
  → 偏差>30%扣5分，>50%扣10分
□ 指令对齐：目标 / 事件 / 伏笔，按写作指令完全落实了吗？
□ 大纲对齐：本章有效推动了整体剧情吗？
```

## 伏笔兑现验证

```
□ 核心条件：伏笔要求的触发条件是否满足？
□ 触发对象：伏笔说的对象是否被正确对待？
□ 角色反应：反应是否合理适度？
□ ⚠️ 伏笔逻辑硬伤 → 直接退回，总分 ≤ 60分
```

## 五层红线核对

触发任意带*项，必须退回！

```
□ *字数偏差：是否超出目标50%以上？
□ *设定/数值：点数、资源、等级与上一章状态卡严格一致吗？
□ *状态卡矛盾：本章内容是否与上一章状态卡描述矛盾？
□ *逻辑/智商：反派智商在线吗？主角是否存在未卜先知？
□ *伏笔逻辑：伏笔兑现是否合理？触发条件是否满足？
□ *毒点检测：有无严重憋屈、圣母双标、上帝说教？
□ *AI味核查：排除了连续排比句、"冷笑/嘴角微扬"复读吗？
□ *末尾钩子：结尾留下了悬念、危机或大反转吗？
```

## 状态提取确认（仅通过时）

```
□ 如果决定给 90 分以上通过，本章显式发生的数值/状态变化，
  准备好 JSON 写入 chapter_state 了吗？
```

## 任务完成检查

```
□ 是否已执行 task_complete？
  没有这一步，调度器永远认为任务还在运行！
```

## 死刑机制检查

```
□ 机械降神：是否有未铺垫的神秘人物/神器救场？
□ AI烂梗：是否有"冷笑"、"嘴角微扬"、"眼中闪过寒芒"？
□ 上帝视角：是否有"伏笔已记录"、"剧情节点"等破壁词汇？
□ 触发死刑 → 总分50分以下，立即退回
```
