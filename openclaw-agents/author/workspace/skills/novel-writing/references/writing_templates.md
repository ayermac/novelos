# 写作模板参考

本文档包含各类写作模板，用于提升写作质量、避免 AI 烂词、控制节奏。

---

## 一、死刑红线

### AI 烂词（出现即 50 分）

以下词汇出现 = 立即退回：

**表情类**：
- ❌ 冷笑
- ❌ 嘴角微扬、嘴角勾起
- ❌ 眯起眼睛
- ❌ 眼中闪过一丝/一道
- ❌ 眼神复杂
- ❌ 深吸一口气
- ❌ 倒吸一口凉气
- ❌ 瞳孔骤缩/收缩
- ❌ 目光闪烁

**氛围类**：
- ❌ 沉默片刻
- ❌ 气氛变得
- ❌ 一股
- ❌ 似乎
- ❌ 仿佛

### 替代表达

**眼神变化**：
- ✅ 瞳孔收缩、目光一凝、眉头紧锁
- ✅ 目光锐利、视线锐利

**笑容**：
- ✅ 咧嘴、咧开嘴、龇牙
- ✅ 皮笑肉不笑
- ✅ 笑容凝固

**情绪描写**：
- ✅ 用行动代替——握拳、后退、转向
- ✅ 用对话标签——他说、他道、他问

**确定性表达**：
- ❌ "仿佛"一阵风吹过
- ✅ 风吹过
- ❌ "似乎"有人来了
- ✅ 有人来了

## 三、对话模板

### 愤怒

```
【愤怒对话模板】

说话人特征：
- 语气：冷硬、压迫
- 动作：握拳、前进、瞪视
- 心理：被触犯底线、不可容忍

对话格式：
"{dialogue}。"他{action}，目光{expression}。

示例：
"你觉得我会放过你？"他一步步逼近，每一步都像踏在对方心跳上。
```

### 平静

```
【平静对话模板】

说话人特征：
- 语气：平稳、从容
- 动作：端茶、翻页、远眺
- 心理：掌控局势、胸有成竹

对话格式：
"{dialogue}。"他{action}，仿佛{analogy}。

示例：
"风起云涌，不过是过眼云烟。"他抿了一口茶，目光越过窗外的山峦。
```

### 紧张

```
【紧张对话模板】

说话人特征：
- 语气：短促、有力
- 动作：握紧武器、后退、环顾
- 心理：警惕、防备

对话格式：
"{short_dialogue}。"他{action}，"{follow_up}。"

示例：
"不对劲。"他突然压低声音，"有埋伏。"
```

### 兴奋

```
【兴奋对话模板】

说话人特征：
- 语气：轻快、上扬
- 动作：拍肩、前进、展颜
- 心理：期待、激动

对话格式：
"{dialogue}！"他{action}，眼中{expression}。

示例：
"这次一定能成功！"他兴奋地拍了拍同伴的肩膀，眼中满是期待。
```

### 悲伤

```
【悲伤对话模板】

说话人特征：
- 语气：低沉、缓慢
- 动作：垂下视线、握紧、沉默
- 心理：失望、无力

对话格式：
"{dialogue}。"他{action}，声音{voice_state}。

示例：
"有些事，终究无法改变。"他垂下视线，声音像是从很远的地方传来。
```

## 四、节奏控制模板

### 开篇

```
【开篇节奏模板】

目标：快速吸引读者，建立期待

结构：
1. 钩子（50字内）：悬念/冲突/怪事
2. 环境（100字）：时间地点人物
3. 冲突（200字）：打破平衡
4. 目标（100字）：本章要达成什么

禁忌：
- ❌ 大段背景介绍
- ❌ 慢热的景物描写
- ❌ 从睡觉/起床开始

示例钩子：
- "三天后，宗门大比。"
- "没有人知道他来自哪里。"
- "那把剑，他等了十年。"
```

### 高潮

```
【高潮节奏模板】

目标：情绪爆发，张力拉满

结构：
1. 铺垫（20%）：最后准备，一触即发
2. 爆发（50%）：高潮事件，情绪顶点
3. 转折（20%）：意外变数，打破预期
4. 收尾（10%）：余波，引向下章

技巧：
- 短句加速节奏
- 动作描写密集
- 减少心理独白
- 留悬念不解答

节奏控制：
- 200字/事件点
- 每段不超过3行
- 对话+动作交织
```

### 结尾

```
【结尾节奏模板】

目标：留下钩子，制造期待

结构：
1. 收束（60%）：本章事件收尾
2. 铺垫（30%）：下一章预告
3. 钩子（10%）：悬念/转折/怪事

钩子类型：
- 悬念钩子：留下未解之谜
- 转折钩子：打破读者预期
- 期待钩子：预告重要事件
- 威胁钩子：暗示危机来临

示例：
- "他不知道，身后的影子已经动了。"
- "三天后，就是那个日子。"
- "这一战，才刚刚开始。"
```

### 过渡

```
【过渡节奏模板】

目标：承上启下，调节阅读节奏

结构：
1. 回顾（20%）：上章关键信息
2. 状态（30%）：人物当前状态
3. 目标（30%）：下一步计划
4. 推进（20%）：进入下一事件

技巧：
- 长句舒缓节奏
- 增加心理描写
- 环境细节丰富
- 对话简洁

禁忌：
- ❌ 纯信息堆砌
- ❌ 无目的闲聊
- ❌ 重复已知信息
```

## 五、情绪模板

### 期待

```
【期待情绪模板】

目标：让读者期待后续发展

触发条件：
- 预告重要事件
- 暗示隐藏力量
- 铺设未解之谜

描写要点：
- 时间节点（如"三天后"）
- 关键物品（如神秘玉佩）
- 人物期待（如眼神热切）

示例：
"三天后，宗门大比正式开始。"
"他已经迫不及待想看看，那传闻中的剑法究竟有多强。"
"所有人都知道，这一战将改变一切。"
```

### 紧张

```
【紧张情绪模板】

目标：让读者感到压迫和危机

触发条件：
- 危险迫近
- 时间压力
- 实力差距

描写要点：
- 短促有力
- 动作密集
- 心跳加速

示例：
"气息越来越近。"
"三息。他只剩三息的时间。"
"汗水顺着额角滑落，他握剑的手微微发紧。"
```

### 爽快

```
【爽快情绪模板】

目标：让读者感到痛快和解气

触发条件：
- 打脸反派
- 实力碾压
- 扬眉吐气

描写要点：
- 反差对比
- 干脆利落
- 旁观反应

示例：
"一剑。仅一剑，胜负已分。"
"全场寂静。没有人想到，结果会是这样。"
"他收回剑，目光扫过那些曾经轻视他的面孔。"
```

### 悲伤

```
【悲伤情绪模板】

目标：让读者产生共情和感动

触发条件：
- 重要人物离去
- 理想破灭
- 无能为力

描写要点：
- 细节触动
- 环境烘托
- 心理独白

示例：
"雨还在下。"
"他站在墓前，久久没有说话。有些话，再也无人可说。"
"风吹过，带走了最后一片落叶，也带走了最后的念想。"
```

## 六、动作描写模板

### 出剑

```
【出剑动作模板】

层次：
1. 准备：姿势、气息、眼神
2. 出手：速度、轨迹、气势
3. 结果：效果、反应、环境

示例：
他脚踏七星，剑随身走。
剑光一闪，快得让人来不及反应。
待众人回过神，剑已归鞘，对手的衣襟裂开一道细线。
```

### 遁走

```
【遁走动作模板】

层次：
1. 决断：判断局势、下定决定
2. 执行：手段、速度、掩护
3. 结果：成功、损失、追兵

示例：
局势不对。他立刻做出判断。
一道烟雾炸开，他的身影消失在原地。
待烟雾散去，只剩下一地残影。
```

## 七、使用方式

### 在 SKILL.md 中引用

```markdown
写作前必读：
- references/writing_templates.md 第一、二节（死刑红线、场景模板）
- references/checklist.md（自检清单）
```

### 命令行查询

```bash
# 获取上下文（包含死刑红线）
python3 tools/db.py build_context <project> <chapter> author

# 健康报告
python3 tools/db.py health_report <project>
```

### 写作时对照

1. 写完每个场景 → 对照场景模板检查结构
2. 写完对话 → 对照对话模板检查语气
3. 写完章节 → 对照死刑红线检查禁词
4. 发布前 → 对照节奏模板检查开头结尾
