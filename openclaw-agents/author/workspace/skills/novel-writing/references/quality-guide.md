# 质检防退回指南

质检 Agent 极其严格。遵循以下准则，避免稿件被退回重写。

---

## 一、对抗"AI写作惯性"

大语言模型在创作小说时有固定的坏习惯，必须刻意避开：

### 1. 严禁动作/神态复读

**禁止使用的烂梗：**
- 冷笑
- 嘴角微扬
- 眼中闪过一道寒光/冷芒
- 倒吸一口凉气
- 眉头紧锁
- 眼神一凛

**替代方案：** 用具体的动作细节

```
❌ 他冷笑一声。

✅ 他把手里的笔重重扣在桌面上。
✅ 他没说话，只是看着窗外，过了很久才开口。
```

### 2. 严禁空洞修辞与排比

**禁止使用：**
- 星辰大海
- 令人窒息
- 不仅...而且...
- 既是...又是...

**原因：** 网文需要的是画面感，不是公文宣讲。

### 3. 严禁上帝视角说教

**绝对不要**在章节末尾或段落结尾，用作者的口吻去总结大道理。

```
❌ 这就是职场的残酷……

❌ 或许，在这个末世里，每个人都在为自己的选择付出代价。

✅ 让角色自己去感受，不要替读者总结！
```

### 4. Show, Don't Tell

```
❌ 两人爆发了激烈的争吵。

✅ （直接写出他们互相攻击的恶毒对话）
```

---

## 二、逻辑与智商对齐

### 反派必须带脑子

严禁通过把反派写成"弱智"来衬托主角的牛逼。

**主角赢的原因应该是：**
- 准备更充分
- 底牌更多
- 抓住了信息差

**而不是：**
- 高管突然犯低级错误
- 老狐狸莫名其妙降智

### 主角不能全知全能

主角只能基于【上一章状态卡】和【已有信息】做决策。

---

## 三、节奏控制与爽点分布

| 阶段 | 占比 | 目标与写法 |
|------|------|-----------|
| 开场 | 10% | 承接上一章钩子，前3段内必须出现动作或对话 |
| 发展 | 40% | 每500字给一次正向微反馈（获取信息、被人惊叹、数值提升） |
| 高潮 | 30% | 核心冲突爆发，必须有"压抑→爆发"的反转过程 |
| 收尾 | 20% | **必须留钩子！禁止平淡收尾！** |

---

## 四、结尾钩子强制规范

章节最后一段，必须属于以下三种情况之一：

### 1. 悬念钩
角色发现了一个惊人的异常物品/数据/信息，章节戛然而止。

### 2. 危机钩
主角刚松一口气，更大的致命威胁突然降临/敲门/来电。

### 3. 期待钩
主角终于拿到了核心道具/准备好反击方案，说出一句宣战的话，下一章即将大开杀戒。

---

## 五、快速检查清单

提交前自我审视：

**基础与设定**
- [ ] 字数达标（严格控制在指令要求的区间内）？
- [ ] 数值来源：本章数值变化完全基于 `chapter_state`？
- [ ] 伏笔处理：指令要求埋设/兑现的伏笔，是否已融入正文？

**质量红线**
- [ ] 智商在线：反派的行为符合其身份和利益吗？
- [ ] 破局合规：解决困难是基于逻辑推演，不是"神秘NPC/机械降神"？
- [ ] 去除AI味：删掉所有的"冷笑/微扬/倒吸凉气"了吗？
- [ ] 杜绝说教：章节末尾没有假大空的人生总结吧？
- [ ] 结尾强制：最后一段有悬念/危机/期待钩子吗？

**修改模式额外确认**
- [ ] 质检痛点：质检报告中的 `issues` 是否已被彻底抹除？
- [ ] 精准执行：质检的 `suggestions` 是否已经完全落实？

**任务完成检查**
- [ ] 是否已执行 `task_complete`？（没有这步，调度器永远认为任务还在运行）
