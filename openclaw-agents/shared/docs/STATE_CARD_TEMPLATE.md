# 状态卡模板

## 概述

状态卡（chapter_state）用于记录每章结束时的数值状态，防止长篇创作中的数值漂移。

---

## 标准结构

```json
{
  "数值类": {
    "金钱": 10000,
    "修为": 1500,
    "等级": "筑基初期",
    "声望": 50,
    "人气": 30
  },
  "位置类": {
    "当前位置": "青云宗·外门",
    "所属势力": "青云宗"
  },
  "角色类": {
    "已登场角色": ["林默", "赵婉清", "赵国栋"],
    "关系变化": {
      "赵婉清": "好感+10（当前：45）",
      "赵国栋": "敌意+5（当前：-20）"
    }
  },
  "伏笔类": {
    "已埋设": ["P001", "P003"],
    "已兑现": ["P002"],
    "本章提及": ["P001"]
  },
  "时间类": {
    "故事时间": "第3天·下午",
    "距下一事件": "还有2天"
  },
  "剧情类": {
    "当前主线": "宗门大比筹备",
    "当前支线": "与赵国栋的矛盾",
    "待解决冲突": ["赵国栋的刁难"]
  }
}
```

---

## 必填字段

### 1. 数值类

| 字段 | 说明 | 示例 |
|------|------|------|
| `金钱` | 当前持有的货币 | 10000 |
| `修为` | 当前修为值 | 1500 |
| `等级` | 当前境界/等级 | "筑基初期" |

**规则**：
- 必须从上一章状态卡抄写，不得自行编造
- 如果有变化，必须明确标注变化原因

### 2. 位置类

| 字段 | 说明 | 示例 |
|------|------|------|
| `当前位置` | 角色所在地点 | "青云宗·外门" |
| `所属势力` | 当前归属 | "青云宗" |

### 3. 伏笔类

| 字段 | 说明 | 示例 |
|------|------|------|
| `已埋设` | 本章新埋设的伏笔代码 | ["P001", "P003"] |
| `已兑现` | 本章兑现的伏笔代码 | ["P002"] |
| `本章提及` | 本章提及但未兑现的伏笔 | ["P001"] |

---

## 可选字段

### 角色关系

```json
{
  "关系变化": {
    "角色名": "变化描述（当前值）"
  }
}
```

### 时间线

```json
{
  "故事时间": "第N天·时段",
  "距下一事件": "还有X天"
}
```

### 剧情进度

```json
{
  "当前主线": "主线描述",
  "当前支线": "支线描述",
  "待解决冲突": ["冲突1", "冲突2"]
}
```

## 使用方法

### Editor 写入状态卡

```bash
python3 tools/db.py chapter_state xuanhuan 10 --set '{
  "数值类": {
    "金钱": 12000,
    "修为": 1800,
    "等级": "筑基中期"
  },
  "位置类": {
    "当前位置": "青云宗·内门",
    "所属势力": "青云宗"
  },
  "伏笔类": {
    "已埋设": ["P004"],
    "已兑现": ["P001"],
    "本章提及": []
  }
}' "第10章：林默突破筑基中期，获得内门资格"
```

### Author 读取状态卡

```bash
python3 tools/db.py chapter_state xuanhuan 10
```

---

## 注意事项

### 禁止事项

1. ❌ **禁止自行编造数值** - 必须基于上一章状态卡
2. ❌ **禁止跳过状态卡** - 每章必须读取
3. ❌ **禁止忽略伏笔类** - 必须记录本章处理的伏笔

### Editor 职责

- 质检通过后，必须写入状态卡
- 状态卡摘要必须包含关键变化
- 数值变化必须有明确依据

### Author 职责

- 创作前必须读取上一章状态卡
- 数值从状态卡抄写，不得凭记忆
- 如发现状态卡缺失，应通知 Planner

---

## 状态卡历史

状态卡每次写入都会自动记录到 `state_history` 表：

```bash
# 查看状态变更历史
python3 tools/db.py state_history xuanhuan

# 查看特定字段的变化
python3 tools/db.py state_history xuanhuan 金钱
```
