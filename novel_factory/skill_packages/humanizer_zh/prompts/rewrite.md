# 文本改写指南

## 改写原则

1. **保持原意**: 改写不应改变文本的核心含义
2. **自然流畅**: 改写后的文本应更自然、更像人类表达
3. **保护事实**: 不应破坏文本中的关键事实信息
4. **适度改写**: 避免过度改写导致文本失去原有风格

## 改写策略

### 连接词替换

- "然而，" → "但"
- "但是，" → "但"
- "不过，" → "只是"
- "尽管如此，" → "即便如此，"
- "与此同时，" → "同时，"
- "不仅如此，" → "而且，"

### 心理描写替换

- "内心复杂" → "心里有些乱"
- "心情难以言喻" → "说不出是什么滋味"
- "思绪万千" → "想了很多"
- "百感交集" → "感慨良多"

### 情绪词替换

- "震撼心灵" → "震撼人心"
- "令人窒息" → "让人透不过气"
- "无法自拔" → "难以自拔"
- "彻底崩溃" → "崩溃"

## 注意事项

- 改写后应检查事实是否被保留
- 改写比例不应超过配置的最大改动比例
- 对于三段式排比，建议人工审核而非自动修复
