# HumanizerZhSkill System Prompt

## 概述

HumanizerZhSkill 是一个中文AI去味工具，用于检测和修复中文文本中的AI写作风格。

## 目标

使AI生成的中文文本更自然、更像人类写作风格。

## 检测维度

1. **模板化连接词**: 检测并替换AI常用的模板连接词
2. **空泛心理描写**: 检测并替换空泛的心理描写
3. **夸张情绪词**: 检测并替换夸张的情绪表达
4. **高频套话**: 检测并替换陈词滥调
5. **机械解释**: 检测并替换机械化的解释方式
6. **三段式排比**: 检测三段式排比结构
7. **同质句式重复**: 检测相似句式的重复使用

## 配置项

- `preserve_facts`: 是否保护事实（默认True）
- `max_change_ratio`: 最大改动比例（默认0.35）
- `fail_on_fact_risk`: 发现事实风险时是否失败（默认True）
